PROYECTO FINAL – PROGRAMACIÓN III
Universidad Mariano Gálvez – 2025

Nombre del proyecto:
Sistema de Gestión de Tarjetas de Crédito

Lenguaje y entorno de desarrollo:
C# con .NET 7.0, desarrollado en Visual Studio 2022.

---

INSTRUCCIONES PARA EJECUTAR EL PROYECTO:

1. ABRIR VISUAL STUDIO
   - Abrir Visual Studio 2022 o superior.

2. CLONAR EL REPOSITORIO O DESCARGARLO
   - Desde GitHub:
     https://github.com/JosselynSamayoa/Proyecto-Final-2025.git
   - O descargar el ZIP desde GitHub y extraerlo en tu computadora.

3. ABRIR LA SOLUCIÓN
   - Buscar el archivo:
     Proyecto_Final_Sistema_Bancario.sln
   - Hacer doble clic o abrir desde Visual Studio: `Archivo → Abrir → Proyecto/Solución`

4. RESTAURAR PAQUETES Y COMPILAR
   - Esperar a que Visual Studio restaure los paquetes automáticamente.
   - Si no lo hace, ir al menú: `Compilar → Compilar solución`

5. EJECUTAR EL PROYECTO
   - Seleccionar la opción: `Inicio (F5)` o hacer clic en el botón de ejecutar en Visual Studio.

---

REQUISITOS:

- Tener instalado Visual Studio 2022 o superior.
- .NET 7.0 SDK instalado.
- Acceso a Internet para restaurar paquetes NuGet, si es la primera vez que se abre.

---

INFORMACIÓN ADICIONAL:

Este sistema gestiona tarjetas de crédito simuladas, permitiendo operaciones básicas mediante una arquitectura con controladores, modelos y estructura de datos.
